# Dosya Yönetim Projesi

## Proje Açıklaması
Bu proje, kullanıcı tarafından talep edilen "sil bu dosyayı" işlemi için oluşturulmuş temel bir dosya yönetim sistemidir.

## Özellikler
- Dosya silme işlevselliği
- Dosya listeleme
- Modüler yapı
- Temel CSS stilleri

## Klasör Yapısı
```
project/
├── src/
│   └── main.py
├── utils/
│   └── file_manager.py
├── assets/
│   └── style.css
├── requirements.txt
└── README.md
```

## Kurulum
1. Projeyi klonlayın
2. Python 3.8 veya üstü kurulu olduğundan emin olun
3. Gerekli bağımlılıkları yükleyin

## Kullanım
Ana uygulamayı çalıştırmak için:
```bash
python src/main.py
```
